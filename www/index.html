Défilé Migration Forecast

About the Site

The Défilé de l'Écluse is a renowned birdwatching site located in the Rhône-Alpes region of France. It serves as a natural bottleneck for migrating raptors and other birds, making it an ideal location for observing seasonal movements.

Project Aims

  • Forecast bird migration for upcoming days using deep learning models.
  • Visualize daily migration data for key raptor species.
  • Support ornithologists and birdwatchers with timely insights.
